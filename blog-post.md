# 근로계약서부터 퇴직금 정산까지, 노무서류 18종을 한 번에 — 노무뚝딱

> 직원을 채용하면서 근로계약서는 어떻게 쓰지? 급여명세서는 꼭 줘야 하나? 취업규칙은 뭘 넣어야 하지?
> 중소기업 대표님이라면 한 번쯤 고민해 본 질문들입니다.

**노무뚝딱**은 이런 고민을 해결하기 위해 만든 노무서류 관리 웹앱입니다. 근로계약서, 급여명세서, 임금대장, 취업규칙은 물론 입사부터 퇴사까지 필요한 **노무서류 18종**을 빈칸 채우기처럼 쉽게 만들 수 있습니다.

## 직원 한 번 등록하면 끝

노무뚝딱의 핵심은 **자동 입력**입니다.

1. **회사 정보 설정** — 회사명, 대표자, 사업자번호를 한 번만 입력
2. **직원 등록** — 이름, 주민번호, 급여 정보 입력
3. **서류 작성** — 직원 선택하면 정보가 자동으로 채워짐

계약서를 쓸 때마다 이름, 주민번호, 주소를 다시 입력할 필요가 없습니다. 등록된 직원을 클릭하면 모든 정보가 자동으로 들어갑니다.

## 급여 최적화로 절세까지

같은 월급이라도 구성을 어떻게 하느냐에 따라 세금이 달라집니다.

**노무뚝딱의 급여 최적화 기능**을 사용하면:

| 항목 | 금액 | 세금 여부 |
|------|------|----------|
| 기본급 | 260만원 | 과세 |
| 식대 | 20만원 | **비과세** |
| 자가운전보조금 | 20만원 | **비과세** (차량 보유 시) |
| 보육수당 | 20만원 | **비과세** (6세 이하 자녀) |

총 급여 320만원을 입력하면 비과세 항목을 자동으로 배분합니다.
**월 약 3만원, 연간 약 36만원 절약** 효과!

## 사용 방법 (3단계)

### STEP 1. 회사 정보 설정 (3분)
설정 메뉴에서 회사명, 대표자, 사업자번호, 주소, 전화번호를 입력합니다. 사업자등록증만 있으면 됩니다.

### STEP 2. 직원 등록 (5분/명)
직원관리 메뉴에서 직원 정보와 급여를 입력합니다. **급여 최적화 추천** 버튼을 누르면 비과세 항목이 자동 배분됩니다.

### STEP 3. 서류 작성 & 출력 (2분)
원하는 서류를 선택하고 직원을 클릭하면 자동 입력됩니다. 미리보기로 확인한 후 인쇄 또는 PDF로 저장하세요.

## 왜 노무뚝딱인가요?

- **쉽다** — 빈칸 채우기처럼 간단합니다
- **빠르다** — 직원 등록 후 서류 작성까지 2분
- **정확하다** — 4대보험, 소득세, 퇴직금 자동 계산
- **안전하다** — 데이터는 내 브라우저에만 저장, 서버 전송 없음
- **표준 양식** — 고용노동부 표준계약서·취업규칙 기반
- **절세 효과** — 비과세 항목 자동 최적화

## 자주 묻는 질문

**Q. 저장한 데이터가 사라졌어요**
데이터는 현재 브라우저에만 저장됩니다. 캐시/쿠키 삭제 시 데이터도 삭제되니, 중요한 서류는 PDF로 저장해 두세요. 설정에서 데이터 내보내기(JSON)도 가능합니다.

**Q. 이 서류로 실제 계약해도 되나요?**
고용노동부 표준계약서 기반입니다. 일반적 상황에서 충분히 사용 가능하며, 특수한 상황은 노무사 상담을 권장합니다.

**Q. 모바일에서도 되나요?**
네, 모바일 브라우저에서도 접속 가능합니다. 다만 입력 항목이 많으므로 PC 사용을 권장합니다.
